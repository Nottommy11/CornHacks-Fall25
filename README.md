# CornHacks Fall 2025 Project Repositories

## Node Firmware
**Repository:** [CornHacks_Fall25_Node](https://github.com/Nottommy11/CornHacks_Fall25_Node)  
**Description:** Interfaces with physical components for the hydroponics setup and acts as a node to send the data up the chain to the database.

---

## Pi Controller - API
**Repository:** [CornHacks-Fall25-PiController](https://github.com/Nottommy11/CornHacks-Fall25-PiController)  
**Description:** Acts as a central hub on a network and allows nodes to feed their data up the chain.

---

## API Service
**Repository:** [CornHacks_Fall25_API](https://github.com/Nottommy11/CornHacks_Fall25_API)  
**Description:** Adds an additional layer to access the LLM Wrapper and GraphQL endpoint.

---

## Frontend Dashboard
**Repository:** [CornHacks_Fall25_FrontEnd](https://github.com/Nottommy11/CornHacks_Fall25_FrontEnd)  
**Description:** Provides a dashboard with an overview of data from the hydroponics setup and allows for AI chat with Google Gemini.
